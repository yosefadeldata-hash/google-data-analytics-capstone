## 📊 خطوات العمل (تحليل البيانات)
لقد اتبعت منهجية Google في تحليل البيانات:
1. **Ask:** تحديد الأسئلة التجارية.
2. **Prepare:** جمع وتجهيز البيانات.
3. **Process:** تنظيف البيانات والتحقق من صحتها.
4. **Analyze:** استخراج الأنماط والنتائج.
5. **Share:** عرض النتائج وتصور البيانات.
6. **Act:** تقديم التوصيات لاتخاذ القرار.
