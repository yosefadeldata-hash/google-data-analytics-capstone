# google-data-analytics-capstone
## 🎯 المهمة التجارية (Business Task)
الهدف الأساسي من هذا المشروع هو تحليل بيانات رحلات Cyclistic لفهم الاختلافات السلوكية بين "الركاب العاديين" (Casual Riders) و "الأعضاء السنويين" (Annual Members). 

**الهدف النهائي:** تقديم توصيات قائمة على البيانات لمساعدة فريق التسويق في الشركة على تحويل الركاب العاديين إلى أعضاء سنويين.

## ❓ الأسئلة الرئيسية (Key Questions)
1. كيف تختلف أوقات استخدام الدراجات بين الأعضاء السنويين والركاب العاديين؟
2. ما هي الأنماط السلوكية المختلفة في اختيار المحطات أو مسافات الرحلات؟
3. لماذا قد يقرر الراكب العادي التحول إلى عضو سنوي؟
